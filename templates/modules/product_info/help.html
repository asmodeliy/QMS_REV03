📊 Product Info {% if locale == 'ko' %}개요 - IP 매트릭스{% else %}Overview - IP Matrix{% endif %}

{% if locale == 'ko' %}

Product Info (제품 정보 관리)는 다양한 공정 노드에서 제공 가능한 회사의 모든 IP(Intellectual Property) 포트폴리오를 한눈에 조회하는 시스템입니다.

주요 특징

  • 📊 IP 매트릭스 조회: 원하는 IP의 다양한 노드 (SF28nm, SF14nm, SF8nm, SF5nm, SF4nm 등)에서의 개발 상태를 확인합니다.
  • 🔍 상태 조회: 각 IP-노드 조합의 상태를 분류합니다 (Mass 생산, Proven 검증완료, Dev 개발중, Plan 계획, 또는 미지원).
  • 🏗️ IP 포트폴리오: PHY, 인터페이스, 아날로그 회로 등 다양한 IP를 관리합니다.
  • ⚙️ 세부 정보: 각 IP의 성능 사양, 특징, 적용 예시를 확인합니다.
{% else %}

Product Info is a system for viewing the entire portfolio of company IPs (Intellectual Property) available across different process nodes at a glance.

Key Features

  • 📊 IP Matrix View: Check development status of desired IP across various nodes (SF28nm, SF14nm, SF8nm, SF5nm, SF4nm, etc.)
  • 🔍 Status View: Classify status of each IP-node combination (Mass, Proven, Dev, Plan, or Not Supported).
  • 🏗️ IP Portfolio: Manage diverse IPs including PHY, interfaces, and analog circuits.
  • ⚙️ Details: View performance specs, features, and application examples for each IP.
{% endif %}

🔍 {% if locale == 'ko' %}IP 검색 및 필터링{% else %}IP Search & Filtering{% endif %}

{% if locale == 'ko' %}IP 검색 팁{% else %}IP Search Tips{% endif %}

  • 📝 IP 이름 검색: "MIPI", "PHY", "Ethernet" 등 정확한 이름 또는 부분 검색
  • 🔤 인터페이스 표준 검색: MIPI, PCIe, USB, SATA 등의 표준명으로 검색
  • ⚡ 실시간 검색: 메인 페이지의 IP 목록에서 바로 필터링

노드별 조회

  • SF28nm: 첫 세대 공정, 기존 IP 대부분 지원
  • SF14nm: 주력 공정, 신규 IP 추가 지원
  • SF8nm: 고성능 공정, 검증 완료된 IP들
  • SF5nm: 최신 공정, 개발 중인 고급 IP
  • SF4nm: 차세대 공정, 향후 계획 IP

상태별 조회

  • 🟢 Mass (대량생산): 현재 상용 제품으로 사용 가능
  • 🟡 Proven (검증완료): 기술 검증 완료, 즉시 사용 가능
  • 🔵 Dev (개발중): 개발 진행 중, 최근에 사용 가능
  • ⚫ Plan (계획): 향후 개발 계획
  • ⚪ 미지원: 해당 노드에서 미지원

정렬 옵션

  • IP 이름순
  • 인터페이스 표준별
  • 해당 노드에서의 첫 지원 시점
