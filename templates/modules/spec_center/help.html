📂 Spec Center 개요

Spec Center (사양 센터)는 회사 내 모든 기술 문서, 사양서, 드로잉을 한곳에서 관리하고 검색하는 중앙 저장소입니다.

주요 특징

  • 📁 카테고리별 정리: 폴더 트리 구조로 문서를 체계적으로 관리합니다.
  • 🔍 빠른 검색: 파일명으로 빠르게 찾을 수 있습니다.
  • 👁️ 인라인 미리보기: PDF와 이미지를 브라우저에서 바로 확인합니다.
  • 📥 쉬운 다운로드: 한 번의 클릭으로 파일을 다운로드합니다.
  • 🏷️ 메타데이터: 파일 크기, 업로드자, 수정 날짜를 표시합니다.
  • 🤖 AI 챗봇: 문서 내용을 기반으로 질문에 답변합니다.

🚀 빠른 시작

1단계: 문서 찾기

  • 방법 1 - 검색 사용 (권장):
    • 상단의 검색창에 찾는 파일명을 입력합니다.
    • 검색 결과에서 원하는 파일을 클릭합니다.
    • 자동으로 해당 폴더로 이동합니다.
  • 방법 2 - 카테고리 탐색:
    • 좌측 카테고리 트리에서 폴더를 클릭합니다.
    • 해당 폴더 내의 모든 파일이 표시됩니다.

2단계: 문서 미리보기

  • PDF 파일: 브라우저에서 바로 열려서 페이지 넘김, 확대/축소가 가능합니다.
  • 이미지 파일: 크기가 크면 다운로드 링크로 표시됩니다.
  • Office 파일 (Excel, Word): 링크가 표시되므로 다운로드 후 열어봅니다.

3단계: 다운로드

  • 파일 행의 우측에 있는 "⬇️ 다운로드" 버튼을 클릭합니다.
  • 파일이 다운로드됩니다.

🔍 검색 및 조회

검색 기능

  • 📝 파일명 검색: 파일명의 일부를 입력하면 자동완성으로 관련 파일을 제안합니다.
  • ⚡ 실시간 검색: 입력할 때마다 결과가 업데이트됩니다.
  • 🎯 정확한 매칭: 검색 결과를 클릭하면 해당 폴더로 이동합니다.

카테고리 탐색

  • 좌측 패널의 폴더 트리를 사용하여 문서를 탐색합니다.
  • 각 폴더를 펼치거나 접을 수 있습니다.
  • 현재 위치는 상단의 경로 표시(Breadcrumb)에 나타납니다.

파일 목록 컬럼

  • 파일명: 클릭하면 미리보기를 확인합니다.
  • 크기: 파일의 용량
  • 업로드자: 파일을 업로드한 사람
  • 수정 날짜: 마지막 수정 시간
  • 다운로드: 파일 다운로드 버튼

⭐ 주요 기능

1. PDF/이미지 인라인 미리보기

  • PDF 파일을 클릭하면 브라우저에서 바로 열립니다.
  • 다음/이전 페이지 네비게이션, 확대/축소, 검색 기능이 있습니다.
  • 이미지도 미리보기를 지원합니다.

2. 클립보드 복사

  • 파일명 옆의 복사 아이콘을 클릭하면 파일명을 클립보드에 복사합니다.
  • 다른 문서나 메일에 쉽게 붙여넣을 수 있습니다.

3. AI 챗봇 지원

  • 우측 하단의 채팅 아이콘을 클릭합니다.
  • 문서 내용에 대한 질문을 입력하면 AI가 답변합니다.
  • 예: "DFMEA는 언제 수행되나요?", "제어계획의 샘플링 방법은?"

4. 모바일 지원

  • 스마트폰/태블릿에서도 Spec Center를 접근할 수 있습니다.
  • 반응형 디자인으로 자동으로 화면 크기에 맞춥니다.

⚙️ 관리자 도구

문서 업로드 (관리자 전용)

  • 관리자는 "+ Upload" 버튼으로 새 파일을 업로드할 수 있습니다.
  • 지원 포맷: PDF, DOC, XLS, PPT, 이미지 등
  • 최대 파일 크기는 시스템 설정에 따릅니다.

폴더 관리 (관리자 전용)

  • 폴더 생성: "New Folder" 버튼으로 새 카테고리를 만듭니다.
  • 폴더명 수정: F2 키 또는 우측 메뉴에서 수정합니다.
  • 폴더 삭제: 우측 메뉴의 삭제 옵션을 사용합니다.

권한 관리 (관리자 전용)

  • 기본적으로 모든 인증된 사용자가 문서를 볼 수 있습니다.
  • 특정 폴더에 접근 제한을 설정할 수 있습니다.
  • 관리자에게 요청하여 권한을 조정합니다.

❓ 자주 묻는 질문

Q: 문서가 최신인지 어떻게 알 수 있나요?

A: 파일 목록의 "수정 날짜" 컬럼을 확인하세요. 최신 문서가 상단에 표시되도록 정렬할 수 있습니다.

Q: 버전이 여러 개 있는 문서를 찾으려면?

A: 파일명에 버전을 포함시킵니다 (예: "DFMEA_ProductXYZ_v1.2.pdf"). 검색할 때 버전을 포함해서 검색하세요.

Q: 문서를 업로드하려면?

A: 현재는 관리자만 업로드 권한이 있습니다. 새 문서를 추가해달라고 관리자에게 요청하세요.

Q: 오프라인에서 Spec Center를 사용할 수 있나요?

A: 아니오, Spec Center는 웹 기반 서비스입니다. 필요한 문서를 미리 다운로드하여 오프라인에서 사용하세요.

Q: 문서 다운로드가 안 됩니다.

A: 브라우저 다운로드 설정을 확인하거나, 관리자에게 권한을 요청하세요. 큰 파일의 경우 시간이 걸릴 수 있습니다.

Q: AI 챗봇이 잘못된 답변을 했어요.

A: AI는 학습 데이터 기반으로 답변합니다. 명확하지 않으면 문서를 직접 확인하거나 팀에 물어보세요.

📂 Spec Center overview

Spec Center

Central hub for specs, drawings, and reference docs with in-browser previews and category tree.

Preview
  • Navigate: Category tree + breadcrumb
  • Preview: PDF/images inline; large Office files show download link
  • Download: Icon on each row

🔎 Browse & search

  • Typeahead search: Use the header search to find files by name; click result to jump to its folder.
  • Columns: Size, uploader, and updated-at help validate freshness before opening.
  • Keyboard: F2 renames a category (admin only), Enter opens selected file.

⚙️ Admin actions

Upload

Drag files into a category or use the Upload button. Allowed: pdf, pptx, docx, doc, xlsx, xls.

Move

Drag a file row to another category. The tree highlights valid drop targets.

Rename

Select a category and press F2 to inline-rename; Enter to save, Esc to cancel.

Delete

Use the trash icon on a file or category. Deletions are immediate—double-check before confirming.
