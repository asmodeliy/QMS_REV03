📂 Spec Center {% if locale == 'ko' %}개요{% else %}Overview{% endif %}

{% if locale == 'ko' %}

Spec Center (사양 센터)는 회사 내 모든 기술 문서, 사양서, 드로잉을 한곳에서 관리하고 검색하는 중앙 저장소입니다.

주요 특징

  • 📁 카테고리별 정리: 폴더 트리 구조로 문서를 체계적으로 관리합니다.
  • 🔍 빠른 검색: 파일명으로 빠르게 찾을 수 있습니다.
  • 👁️ 인라인 미리보기: PDF와 이미지를 브라우저에서 바로 확인합니다.
  • 📥 쉬운 다운로드: 한 번의 클릭으로 파일을 다운로드합니다.
  • 🏷️ 메타데이터: 파일 크기, 업로드자, 수정 날짜를 표시합니다.
  • 🤖 AI 챗봇: 문서 내용을 기반으로 질문에 답변합니다.
{% else %}

Spec Center is a centralized repository for managing and searching all technical documents, specifications, and drawings within the company.

Key Features

  • 📁 Organized by Category: Systematically manage documents in a folder tree structure.
  • 🔍 Fast Search: Quickly find files by filename.
  • 👁️ Inline Preview: View PDFs and images directly in the browser.
  • 📥 Easy Download: Download files with one click.
  • 🏷️ Metadata: Display file size, uploader, and modification date.
  • 🤖 AI Chatbot: Answer questions based on document content.
{% endif %}

🚀 {% if locale == 'ko' %}빠른 시작{% else %}Quick Start{% endif %}

{% if locale == 'ko' %}1단계: 문서 찾기{% else %}Step 1: Find Documents{% endif %}

{% if locale == 'ko' %}
  • 방법 1 - 검색 사용 (권장):
    • 상단의 검색창에 찾는 파일명을 입력합니다.
    • 검색 결과에서 원하는 파일을 클릭합니다.
    • 자동으로 해당 폴더로 이동합니다.
  • 방법 2 - 카테고리 탐색:
    • 좌측 카테고리 트리에서 폴더를 클릭합니다.
    • 해당 폴더 내의 모든 파일이 표시됩니다.

2단계: 문서 미리보기

  • PDF 파일: 브라우저에서 바로 열려서 페이지 넘김, 확대/축소가 가능합니다.
  • 이미지 파일: 크기가 크면 다운로드 링크로 표시됩니다.
  • Office 파일 (Excel, Word): 링크가 표시되므로 다운로드 후 열어봅니다.

3단계: 다운로드

  • 파일 행의 우측에 있는 "⬇️ 다운로드" 버튼을 클릭합니다.
  • 파일이 다운로드됩니다.
{% else %}
  • Method 1 - Search (Recommended):
    • Enter the filename to find in the search box at the top.
    • Click the file from search results.
    • Automatically navigate to the containing folder.
  • Method 2 - Browse Categories:
    • Click folders in the category tree on the left.
    • All files in that folder are displayed.

Step 2: Preview Documents

  • PDF Files: Open directly in browser with page navigation and zoom.
  • Image Files: Large images show as download links.
  • Office Files (Excel, Word): Links shown; download and open.

Step 3: Download

  • Click the "⬇️ Download" button at the right of the file row.
  • File will be downloaded.
{% endif %}
